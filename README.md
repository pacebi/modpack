# Pacebi modpack
Modpack for the Pacebi SMP.

## Requirements
- Java 8
- Minecraft Forge 1.16.5
- A reasonably ok computer

## Installation
Installation is simple. Simply download the zip archives, move it to your mods folder, and extract.
